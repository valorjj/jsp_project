### 주제 후보들 : 

[1] 인스타그램 같은 프론트엔트 + 완성도가 중요한 SNS  
[2] 공공데이터 활용한 프로젝트 ( 쓰레기통, 택배, 숙박업체, 핸드폰 사용량 etc)  
[3] 주식 혹은 코인 api 를 활용한 거래 사이트   
[4] 택배 통합 조회   

<선정기준>
1. 재미 (=창의성, 신선함, 참신함)  
2. 완성도 (=기간내에 완성할 수 있는가?, 실제 상용되는 프로그램 수준의 80% 까지 만들 수 있는가?)
